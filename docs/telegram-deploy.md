# Деплой на GitHub Pages и открытие в Telegram

## 1. Запушить код и задеплоить

Деплой запускается автоматически при пуше в ветку `main`:

```bash
git add .
git commit -m "Ваше сообщение"
git push origin main
```

Через 1–2 минуты приложение будет доступно по адресу:

**https://merireact.github.io/avtonomera-market-app**

(или ваш `https://<username>.github.io/<repo-name>`, если репозиторий другой — смотрите `homepage` в `package.json`).

---

## 2. Настроить бота в Telegram, чтобы открывалось приложение

Чтобы при открытии приложения из бота подтягивались имя и аватарка пользователя, приложение должно открываться как **Web App** (Mini App), а не как обычная ссылка.

### Шаг 1: Создать бота (если ещё нет)

1. Откройте [@BotFather](https://t.me/BotFather).
2. Отправьте `/newbot`, придумайте имя и username бота.
3. Сохраните выданный **токен** (для серверной проверки `initData` потом).

### Шаг 2: Указать URL Web App в BotFather

1. В BotFather отправьте `/mybots` → выберите вашего бота.
2. **Bot Settings** → **Menu Button** (или **Configure** для кнопки меню).
3. Укажите URL вашего приложения на GitHub Pages, например:
   ```
   https://merireact.github.io/avtonomera-market-app
   ```
   Без слэша в конце или со слэшем — оба варианта обычно работают.

Либо настройте **Web App** через команду BotFather для кнопки в меню бота (если используете встроенное меню).

### Шаг 3: Открывать приложение как Web App из бота

Пользователь должен открывать приложение **из интерфейса бота** (кнопка меню, команда или inline-кнопка с типом `web_app`). Тогда Telegram передаёт в страницу `initData` с данными пользователя (имя, аватарка и т.д.).

- **Кнопка меню:** если в BotFather задан URL для Menu Button, при нажатии на кнопку внизу (например «Открыть каталог») откроется ваш URL как Web App.
- **Inline-кнопка:** в коде бота при отправке сообщения добавьте кнопку с `web_app.url = "https://merireact.github.io/avtonomera-market-app"`.

После этого при открытии приложения из бота в профиле автоматически отобразятся имя и аватарка из Telegram.

---

## 3. Проверить

1. Запушьте в `main`, дождитесь завершения workflow в **Actions**.
2. Откройте в браузере: `https://merireact.github.io/avtonomera-market-app` — должна открыться главная.
3. Откройте бота в Telegram и нажмите кнопку, которая ведёт на приложение (Web App). Откройте раздел «Профиль» — должны подставиться имя и аватарка из вашего аккаунта Telegram.

Если открыть тот же URL в обычном браузере (не из бота), в профиле будет текст: «Откройте приложение из Telegram, чтобы подтянуть профиль» — это ожидаемо.
